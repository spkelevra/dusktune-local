# dusktune

A minimalist dark-mode local music player for Android, Windows, and macOS.

## Features

- **the grid** — A 3×3 grid of your top tracks, updated as you listen. Tap any tile to play instantly.
- **shuffle** — Instantly randomize your top picks with one tap or the backtick (`) shortcut. Also swipe right on the grid.
- **mixes** — Save your current grid layout as a named mix, load it back anytime, or delete old ones. Access via the dedicated Mixes tab. Long-press / right-click any saved mix to enter edit mode and rearrange tiles with two-tap swap.
- **favorites** — Mark songs as favorites and manage them from the Favorites tab with pin/remove overlays. Play counts displayed next to each track.
- **pins** — Lock specific songs into grid tiles so they always stay in place. Right-click "the grid" button or long-press any song tile from Library, Favorites, or Mixes to enter pin mode. You can also long-press / right-click a grid tile directly to pin that song to the grid immediately.

## Navigation Buttons (Home Screen)

| Button | Tap / Click | Long-Press / Right-Click |
|--------|-------------|--------------------------|
| **Tops** | Cycle to the next page of most-played tracks | Reset back to the first page (most played) |
| **Mix** | Open save dialog for current grid as a new mix | Randomly load a saved mix into the grid |
| **The Grid** | Restore pinned grid and scroll to top | Enter pin mode with currently playing song |

## Controls

**Grid area:** Swipe right → shuffle. Swipe left → restore pinned grid (same as tapping "the grid").

**Filter control:** A rotary knob next to the filter label lets you apply a low-pass or high-pass filter. Drag left to engage the low-pass filter (LPF) — attenuates highs. Drag right for the high-pass filter (HPF) — cuts lows. The neutral position at top removes filtering entirely. Tap/double-tap/right-click the knob's dot to reset to neutral.

**Library, Favorites, and Mixes tabs:** Swipe left → go back to home page. Swipe right → navigate to the next tab sequentially (Library → Mixes → Favorites). Swiping right from Favorites stays on that tab; Settings is not reachable via swipe.

**Grid tiles:** Long-press / right-click a tile → enter pin mode for that song. Tap a tile in pin mode → place it in that position.

**Song list tiles (Library, Favorites, Mixes):** Long-press / right-click → open context menu with pin/remove options.

## Desktop Setup (Windows / macOS)

On first launch, add your music folders via the **Settings** tab. Supported formats: MP3, FLAC, WAV, M4A, OGG, AAC, WMA, Opus.

## Getting Started

**Android:** Install the APK → Accept permissions prompt.

**Windows / macOS:** Download the release ZIP, extract, and run the executable. Add your music folders on first launch via Settings.
